Pathfinding Algorithms Visualizer

About the Pathfinding project

The pathfinding project stems from a technical assignment I had to do for a job interview, where I just had to write an algorithm to solve a maze. I found it such a fun project that I thought it would be cool to make a visualizer for the different algorithms there are. Besides, it teaches me more about the different algorithms and Object Oriented Programming in TypeScript.
